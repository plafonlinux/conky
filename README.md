### - Conky для Manjaro Gnome от PLAFON 

#### - Версия: 2020

#### Как установить:

```
01. Клонировать данный репозиторий 
02. Распаковать содержимое архива в .conky
03. sudo pacman -S conky-manger
04. Включить виджеты в Conky Manager
```
